# Markdown Preview

Live Markdown preview with full GitHub Flavored Markdown support. Syntax highlighting, tables, task lists, code blocks, and HTML export.

## Features

- Live preview with GitHub Flavored Markdown (tables, task lists, etc.)
- Syntax highlighting for code blocks
- Export preview as HTML
- Side-by-side or stacked editor and preview
- Paste from docs or READMEs; no account needed

## Why use this tool

Preview READMEs, docs, or comments before committing. GFM support matches what you see on GitHub and many wikis.
